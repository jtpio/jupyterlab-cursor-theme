# jupyterlab-cursor-dark

JupyterLab dark theme with Cursor vibes.

> [!NOTE]
> This theme is heavily inspired by the default Cursor Dark theme, but it is not an official Cursor product.

## Requirements

- JupyterLab >= 4.0.0

## Install

To install the extension, execute:

```bash
pip install jupyterlab-cursor-dark
```

## Uninstall

To remove the extension, execute:

```bash
pip uninstall jupyterlab-cursor-dark
```
